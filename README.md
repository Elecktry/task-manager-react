Aplicação web de gerenciamento de tarefas desenvolvida com **React**, **Vite** e **JavaScript**.

O objetivo do projeto é praticar conceitos fundamentais do React, como componentes, estados, eventos, renderização de listas, filtros e persistência de dados no navegador.

---

## 📌 Status do projeto

🚧 Projeto em desenvolvimento

Atualmente, o projeto possui a estrutura visual inicial da aplicação, incluindo formulário, resumo de tarefas, filtros e lista de tarefas.

## 🚀 Como executar o projeto

### 1. Clone o repositório

```bash
git clone https://github.com/Elecktry/task-manager-react.git
```

### 2. Entre na pasta do projeto

```bash
cd task-manager-react
```

### 3. Instale as dependências

```bash
npm install
```

Caso esteja usando PowerShell e tenha problema com permissão de scripts, utilize:

```bash
npm.cmd install
```

### 4. Inicie o servidor de desenvolvimento

```bash
npm run dev
```

Ou, no PowerShell:

```bash
npm.cmd run dev
```

### 5. Abra no navegador

O Vite irá mostrar um endereço parecido com:

```text
http://localhost:5173/
```
